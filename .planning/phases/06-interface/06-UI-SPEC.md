## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline badge padding |
| sm | 8px | Compact element spacing, input label gap |
| md | 16px | Default element spacing between sidebar widgets |
| lg | 24px | Section spacing between sidebar control groups |
| xl | 32px | Main area section breaks (between chart, metrics table, heatmap) |
| 2xl | 48px | Top-of-page padding above page title |
| 3xl | 64px | Not used in this phase |

Exceptions: Streamlit's `st.spinner()` overlay uses its own internal padding — do not override. Sidebar widget vertical gaps default to Streamlit's built-in 8px gap between elements; increase to `lg` (24px) between logical groups using `st.sidebar.markdown("---")` dividers.

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px | 400 (regular) | 1.5 |
| Label | 12px | 400 (regular) | 1.4 |
| Heading | 20px | 600 (semibold) | 1.2 |
| Display | 28px | 700 (bold) | 1.1 |

Implementation notes:
- Page title ("Stockformer S&P500 Dashboard") uses `st.title()` — Streamlit renders as Display (28px equivalent, bold).
- Section headings ("Equity Curve", "Performance Metrics", "Prediction Heatmap") use `st.subheader()` — renders as Heading (20px equivalent, semibold).
- Sidebar section labels ("Pipeline Settings", "Backtest Parameters") use `st.sidebar.markdown("**Label**")` — renders as Body bold.
- All Plotly chart axis labels and tick text: 12px (Label role), weight 400, color #9CA3AF.
- Plotly chart titles: 14px (Body role), weight 600, color #E5E7EB.

1. "Pipeline Settings" group: config path, output directory

### Main Area (results)

| Component | Streamlit widget | State |
|-----------|-----------------|-------|
| Initial placeholder | `st.info()` with instruction text | Shown before any run |
| Run spinner overlay | `st.spinner()` wrapping subprocess calls | Active during pipeline execution |
| Live log area | `st.empty()` updated line-by-line with subprocess stdout | Active during pipeline execution |
| Equity curve chart | `st.plotly_chart(fig, use_container_width=True)` | Shown after successful run |
| Metrics table | `st.dataframe(df, use_container_width=True)` | Shown after successful run |
| Prediction heatmap | `st.plotly_chart(fig, use_container_width=True)` | Shown after successful run |
| Error message | `st.error(message)` | Shown on pipeline failure |

### Metrics Table Formatting

Use `st.dataframe()` with explicit column config via `st.dataframe(..., column_config={...})`.

| Column | Display format |
|--------|---------------|
| annualized_return | percentage, 2 decimal places (e.g. "12.34%") |
| total_return | percentage, 2 decimal places |
| max_drawdown | percentage, 2 decimal places, negative values in red |
| alpha_annualized | percentage, 2 decimal places |
| sharpe_ratio | decimal, 2 decimal places (e.g. "1.23") |
| beta | decimal, 2 decimal places |
| top_k | integer (e.g. "10") |
| n_days | integer |

NaN values in metrics table: display as "—" (em dash). Use `df.fillna("—")` before passing to `st.dataframe()`.

## Chart Specifications

### Equity Curve Chart

- Chart type: `go.Scatter` lines, one trace per series
- X-axis: date, formatted as "MMM YYYY" tick labels
- Y-axis: cumulative return multiplier starting at 1.0 (e.g. 1.0 = 0%, 1.15 = +15%)
- Y-axis tick format: "x" suffix (e.g. "1.15x") OR percentage relative to start — use "1.15x" format
- Portfolio trace: name "Portfolio", color #4C9BE8, line width 2px, mode "lines"
- SPY trace: name "SPY", color #9CA3AF, line width 1.5px, mode "lines", line dash "dot"
- Hover template: "{date}: Portfolio {portfolio:.3f}x | SPY {spy:.3f}x"
- Legend: shown, position top-left inside chart
- Chart title: "Cumulative Return: Portfolio vs SPY"
- Height: 400px

### Prediction Heatmap

- Chart type: `go.Heatmap` or `px.imshow`
- X-axis: dates (columns of `regression_pred_last_step.csv`), formatted as "YYYY-MM-DD"
- Y-axis: ticker symbols (rows), top-K tickers only (filtered from full CSV)
- Color scale: `[[0, "#DC2626"], [0.5, "#F9FAFB"], [1, "#16A34A"]]` — diverging red-white-green
- Color bar label: "Predicted Score"
- `zmid=0` so colorscale centers at zero
- Hover template: "{ticker} on {date}: {score:.4f}"
- Chart title: "Prediction Scores — Top-{K} Stocks"
- Height: max(300px, K * 24px) capped at 600px — scales with K
- X-axis tick angle: -45 degrees to prevent label overlap

NaN values in heatmap: pass `zmin` and `zmax` symmetrically (e.g. `zmin=-max_abs`, `zmax=max_abs`); NaN cells render as blank (Plotly default). Do not interpolate.

## Interaction States

| State | Trigger | Visual |
|-------|---------|--------|
| Initial | App loads with no prior run | `st.info()` placeholder in main area; Run button enabled |
| Running | User clicks Run | `st.spinner("Running inference and backtest...")` covers main area; live log lines stream below spinner; Run button disabled (via `st.session_state`) |
| Success | Subprocess exits with code 0 | Spinner dismissed; equity curve, metrics table, heatmap render; download button appears in sidebar |
| Error — no checkpoint | Subprocess fails or output CSV missing | `st.error()` with specific message per error type (see Copywriting) |
| Error — no data for date range | Date range produces empty results | `st.error()` with date-range-specific message |

## Streamlit Theme Configuration

Write to `.streamlit/config.toml` at project root:

```toml
[theme]
base = "dark"
backgroundColor = "#0E1117"
secondaryBackgroundColor = "#1E2130"
primaryColor = "#4C9BE8"
textColor = "#E5E7EB"
font = "sans serif"
```

This makes the sidebar (#1E2130), Run button (#4C9BE8), and page background (#0E1117) match the color contract without any custom CSS.
