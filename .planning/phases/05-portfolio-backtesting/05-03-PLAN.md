    yf.download(tickers + ["SPY"], start=date_index[0] - pd.Timedelta(days=10), end=date_index[-1] + pd.Timedelta(days=5), auto_adjust=True, progress=False, threads=True)
Handles MultiIndex columns from yfinance batch download.
Applies ffill then fillna(0) for NaN prices.
Returns prices DataFrame indexed by DatetimeIndex, reindexed to date_index.

**6. run_backtest_loop(pred_df, prices, tickers, top_k_n) -> tuple[list, list]**
Iterates over pred_df.index (date_index dates).
Per day: select_top_k(pred_df.loc[date], top_k_n), build_portfolio_weights(...), compute_daily_return(weight_now, weight_prev, price_returns, fee=0.001).
price_returns for day t = prices.loc[date, tickers].pct_change().loc[date] — download prices with 10-day buffer to ensure day t-1 exists.
Collects portfolio_returns list and spy_daily_returns list.
Returns (portfolio_returns, spy_daily_returns).

**7. save_outputs(output_dir, date_index, portfolio_returns, spy_daily_returns, metrics, top_k_n)**

- equity_curve.png: matplotlib chart using RESEARCH.md Pattern 5 exactly. Both series start at 1.0 using (1+pd.Series(returns)).cumprod(). No plt.show(). fig.savefig(path, dpi=150), plt.close(fig).
- backtest_summary.csv: dict from compute_performance_metrics + top_k and n_days fields, written as 1-row DataFrame.
- backtest_daily_returns.csv: date / portfolio_return / spy_return columns.

**8. main(output_dir=None, top_k=None)**
Argparse: --output_dir (required), --top_k (int, default=10), --tickers_file (optional, default=data/Stock_SP500_2018-01-01_2024-01-01/tickers.txt), --config (optional, default=config/Multitask_Stock_SP500.conf).
