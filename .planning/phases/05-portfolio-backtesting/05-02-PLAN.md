```
select_top_k(scores: pd.Series, k: int) -> pd.Index
    Returns the k tickers with the highest predicted return scores.
    Uses scores.nlargest(k).index — handles ties by pandas default (first occurrence).

build_portfolio_weights(top_k_index: pd.Index, all_tickers: list, k: int) -> pd.Series
    Returns a Series indexed by all_tickers.
    Selected tickers (in top_k_index) = 1/k; all others = 0.0.

compute_daily_return(
    weight_now: pd.Series,
    weight_prev: pd.Series,
    price_returns: pd.Series,
    fee: float = 0.001
) -> float
    turnover = (weight_now - weight_prev).abs().sum()
    cost = turnover * fee
    gross = (weight_now * price_returns).sum()
    return gross - cost
    price_returns must be indexed by the same tickers; use .reindex(weight_now.index).fillna(0)
    to handle missing tickers (delisted stocks get 0 return, not NaN).
```
