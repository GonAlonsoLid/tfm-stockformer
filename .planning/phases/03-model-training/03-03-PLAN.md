Create `scripts/run_inference.py`. This script must be entirely standalone — it imports ONLY from `lib/`, `Stockformermodel/`, and stdlib. It must NOT import from `MultiTask_Stockformer_train.py` (that module executes side effects at import time).

Structure to implement:

**Step 1 — Imports:**
```python
import argparse, configparser, os, math, sys
import numpy as np
import torch
from torch.utils.data import DataLoader

# Safe lib imports (no side effects at import):
sys.path.insert(0, os.path.dirname(os.path.dirname(os.path.abspath(__file__))))
from lib.Multitask_Stockformer_utils import StockDataset, metric, save_to_csv, log_string
from lib.graph_utils import loadGraph
from Stockformermodel.Multitask_Stockformer_models import Stockformer
```

**Step 2 — Two-phase arg parsing:**
First parse `--config` and `--checkpoint` using `parse_known_args()`. Read the config file with `configparser.ConfigParser()`. Then register all config values as argparse defaults (cuda, batch_size, T1, T2, train_ratio, val_ratio, test_ratio, L/layers, h/heads, d/dims, j/level, s/samples, w/wave, and all [file] keys: traffic_file, indicator_file, adj_file, adjgat_file, model_file, alpha_360_dir, output_dir). Call `parser.parse_args()` for final args. If `--checkpoint` was provided, override `args.model_file` with that value.

**Step 3 — Device setup:**
```python
device = torch.device(f"cuda:{args.cuda}" if torch.cuda.is_available() else "cpu")
```

**Step 4 — Verify Phase 2 outputs exist:**
Check that traffic_file, indicator_file, adjgat_file, and alpha_360_dir all exist. Print a helpful error and sys.exit(1) if any are missing. Also check that args.model_file (checkpoint) exists before attempting to load.

**Step 5 — Load graph and dataset:**
```python
adj, adjgat = loadGraph(args)
test_dataset = StockDataset(args, mode='test')
test_loader = DataLoader(test_dataset, batch_size=args.batch_size, shuffle=False)
```

**Step 6 — Build model and load checkpoint:**
```python
infeature = test_dataset.infea
model = Stockformer(infeature, args.h * args.d, 2, 1, args.L, args.h, args.d, args.s, args.T1, args.T2, device).to(device)
model.load_state_dict(torch.load(args.model_file, map_location=device))
model.eval()
```

**Step 7 — Inference loop:**
With `torch.no_grad()`, iterate test_loader batches. Collect classification and regression predictions and labels. After all batches, call `metric()` to compute and print MAE, accuracy, F1. Create output subdirectories with `os.makedirs(..., exist_ok=True)` and call `save_to_csv()` for both regression and classification outputs using the filenames `regression_pred_last_step.csv`, `regression_label_last_step.csv`, `classification_pred_last_step.csv`, `classification_label_last_step.csv`.

**Step 8 — Entry point:**
```python
if __name__ == '__main__':
    main()
```

Wrap everything from step 2 onward in a `main()` function. Print a summary at the end: "Inference complete. Outputs written to {args.output_dir}".

Do NOT add argparse defaults that conflict with training script patterns. Do NOT open any log file or TensorBoard writer — this is inference-only.
