For compute_features():
- ROC: `ta.roc(df['Close'], length=w)` for each window w
- RSI: `ta.rsi(df['Close'], length=w)` for each window w
- MACD: `ta.macd(df['Close'], fast=12, slow=26, signal=9)` — use standard (12/26/9), take only the MACD line (column index 0); name it 'MACD'
- Bollinger Bands: `ta.bbands(df['Close'], length=20)` only for window=20 (standard BB period); extract columns 0 (upper), 2 (lower), compute width=upper-lower
- VOL_ratio: `df['Volume'] / df['Volume'].rolling(w).mean()` for each window w
- Return DataFrame with all columns; do NOT drop NaN rows here (caller handles warmup)

For build_feature_matrix():
- Read each ticker from `{ohlcv_dir}/{ticker}.parquet` using pd.read_parquet
- Call compute_features(df)
- After processing all tickers, find common index (intersection of all DatetimeIndexes)
- Drop first 60 rows from common index (warmup for longest window)
- For each feature column, build a wide DataFrame: index=common_index, columns=tickers
- Return dict {feature_name: wide_df}

For save_feature_csvs(): create features/ directory, save each wide_df as `{feature_name}.csv` with `df.to_csv()`. Verify shape is [T, N] (index=dates, columns=tickers) before saving — if accidentally transposed, raise ValueError.

For compute_label_csv(): pivot close prices to [T, N], compute `close.shift(-1) / close - 1`, drop last row, fillna(0), save as label.csv.

ANTI-PATTERN TO AVOID: Do NOT save CSVs with stocks as index and dates as columns. Always verify `isinstance(df.index, pd.DatetimeIndex)` before saving.
