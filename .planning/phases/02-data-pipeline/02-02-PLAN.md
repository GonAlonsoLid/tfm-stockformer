Key implementation details:
- `get_sp500_tickers()`: use `pd.read_html("https://en.wikipedia.org/wiki/List_of_S%26P_500_companies")[0]` and return `table['Symbol'].str.replace('.', '-', regex=False).tolist()`
- `download_ohlcv_batched()`: use `yf.download(chunk, start=start, end=end, auto_adjust=True, progress=False)` returning MultiIndex (field, ticker); extract per ticker via `data.xs(ticker, axis=1, level=1)`; add `time.sleep(delay)` after each chunk; on exception retry up to 3× with `time.sleep(delay * (attempt+1))`
- `clean_and_align()`: build master_calendar from the ticker with the most dates (typically SPY-like tickers); for each ticker reindex to master_calendar, ffill(limit=5), then drop ticker if `df.isna().any(axis=1).mean() > max_missing_pct`
- `main()`: use argparse; create `{data_dir}/ohlcv/` directory; for each clean ticker call `df.to_parquet(f'{ohlcv_dir}/{ticker}.parquet')`; write `{data_dir}/tickers.txt` with one ticker per line; print count of saved tickers
